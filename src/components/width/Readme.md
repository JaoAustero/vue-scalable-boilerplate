### Usage
Use the ***:xsmall***, ***:small***, ***:medium***, ***:large***,
***:xlarge*** props to specify the width based on the size of the 
viewport.

```vue
<template>
    <uis-grid class="uis-text-center">
        <uis-width
            :xsmall="'1-1'"
            :small="'1-2'"
            :large="'1-4'"
            :xlarge="'1-6'">
            <div
                class="uis-card
                    uis-card-body
                    uis-card-default">
                    Item
            </div>
        </uis-width>
        <uis-width
```
